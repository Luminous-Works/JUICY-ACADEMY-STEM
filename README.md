# JUICY ACADEMY
## Professional Materials Science & Metallurgy Open-Source Repository

### Overview
JUICY ACADEMY is a centralized resource for STEM educators and materials science students. This repository hosts diverse datasets, high-school curriculum modules, and college-level engineering guides.

### Sub-Repositories / Modules
1.  **Metallurgy Academy**: High-school level introduction to metals, bauxite extraction (Jamaican context), and atomic lattice structures.
2.  **Composites**: College-level materials science curriculum covering smart materials, characterization (SEM/XRD), and selection logic.

### Data Resources
*   **Climate & Weather**: National datasets on climate change and meteorological indicators.
*   **National Statistics**: Population and education access markers.
*   **Seismic & Science**: Science/technology indicators for the Caribbean region.

### Documentation & Lessons
All curriculum developed in collaboration with standards from the National Science Foundation (NSF) and adapted for the Luminous.Works / JUICY ecosystem.
